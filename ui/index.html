TG WS Proxy

Ускоряет Telegram через WebSocket

Приложение поднимает локальный прокси: Telegram подключается к нему, а прокси уже ходит на серверы Telegram по WebSocket. Так обходятся ограничения сети — чаты и медиа открываются быстрее.

Отмена

О программе

TG WS Proxy

Локальный SOCKS5-прокси для Telegram Desktop: перенаправляет трафик через WebSocket на серверы Telegram и ускоряет загрузку сообщений и медиа.

Как это работает

Telegram Desktop → SOCKS5 (127.0.0.1:1080) → TG WS Proxy → WSS (kws*.web.telegram.org) → дата-центр Telegram

  1. На локальном адресе 127.0.0.1:1080 поднимается SOCKS5-прокси.
  2. Прокси принимает только подключения к IP-адресам Telegram и для остального трафика не используется.
  3. Из первого пакета (MTProto obfuscation init) читается номер дата-центра (DC ID).
  4. К этому дата-центру устанавливается WebSocket (TLS) через домены вида kws{N}.web.telegram.org.
  5. Если WebSocket недоступен (например, ответ 302) — соединение идёт обычным TCP до того же DC.

Отличия форка от оригинала

Что изменено в этой сборке

  • Интерфейс на HTML/CSS/JS (WebView) вместо CustomTkinter — современный вид и навигация: Главная, Настройки, О программе, Отличия форка.
  • При запуске сразу открывается главное окно приложения (дашборд), а не только иконка в трее. Кнопка «Свернуть в трей» скрывает окно; из трея пункт «Открыть» снова показывает окно.
  • Живая статистика в трее: в подсказке иконки и в пункте меню отображаются текущие подключения (через WS / через TCP) и объём трафика.
  • Кнопка «Проверить подключение» — проверка каждого DC (WebSocket ✓ или TCP fallback), доступна на главной и в настройках. Результат по каждому DC с цветовой индикацией.
  • Опция «Запускать при входе в Windows» в настройках: включение/выключение применяется сразу, ярлык создаётся или удаляется в папке автозагрузки, без установщика.
  • Единая тема оформления для всех экранов; настройки разбиты на блоки (Прокси, Дата-центры, Дополнительно) с подсказкой формата DC→IP.
  • Свои окна сообщений об ошибках и уведомлений вместо системного MessageBox — в одном стиле с приложением.
  • Закрытие окна по крестику сворачивает приложение в трей, а не завершает его. Выход — через пункт «Выход» в меню треи или кнопку в окне.
  • Первый запуск и настройки интегрированы в одно окно приложения.
